# Governance

This repository is maintained as a governed TSMM-native corpus.

## Authority
Repository maintainers decide package admission, package reclassification, and release readiness.

## Delegation
Contributors may propose packages, edits, examples, and validation improvements through pull requests.

## Enforcement
All changes must preserve:

- package completeness
- schema-valid TSMM graphs
- repository validation passing
- source essay provenance

## Revocation
Packages may be deprecated, superseded, or removed when the modeling stance changes or source provenance is no longer reliable.

## Evidence
Repository validation output is the baseline evidence surface for package integrity.
