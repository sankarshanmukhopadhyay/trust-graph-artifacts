# Intermediary Governance Overlay

**Kind:** overlay  
**Path:** `overlays/intermediary-governance-overlay`  
**Source essay:** The Commons After the Control Plane Shifts  
**Published:** 2026-04-17

## Why this package exists

AI intermediaries become governance chokepoints without carrying governance obligations.

## What the essay contributes

The source essay supplies the design pressure around intermediaries, commons. This package turns that pressure into a TSMM-native shape that can be inspected, tested, and extended.

## What TSMM contributes

TSMM gives this package a structural vocabulary for authority, policy, evidence, trust decisions, and downstream effects.
