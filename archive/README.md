# Archive

This directory holds repository content retained for historical reference but no longer part of the active TSMM-native model.

## Why this exists

The repository previously carried an artifact-first approach and a later binding-oriented transition toward TSMM.
Those materials are preserved here so that prior releases remain inspectable, but they are **not** the active authoring model.

## Active model

The live repository surface is now:

- `profiles/`
- `patterns/`
- `overlays/`
- `systems/`
- `evidence/`
- `provenance/`
- `registries/`
- `schemas/`
- `scripts/`
- `validation/`
- `docs/`

Anything archived under `archive/legacy-artifact-model/` should be treated as superseded structure.
